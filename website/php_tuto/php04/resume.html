Expressions régulières : motifs
^ Début de chaîne $ Fin de chaîne
{n} Le caractère précédent est répété exactement n fois. {n,} Le caractère précédent est répété n ou plus de fois.
{n,m} Le caractère précédent est répété au moins n fois, et au plus m fois. * Le caractère précédent est répété 0 ou plusieurs fois.
+ Le caractère précédent est répété une ou plusieurs fois. ? Le caractère précédent est répété 0 ou une fois.
. Remplace n'importe quel caractère. [] Liste de caractères. [abc] : les caractère a ou b ou c. [a-z] : les caractères de a à z.
() Encadrent une sous-chaîne. | Opérateur ou.
\b Début ou une fin de "mot" \B Inverse de \b
\d Tous les chiffres. \D Inverse de \d.
\s Tous les caractères d'espace (espaces, tabulations, saut de ligne, retour chariot). \w N'importe quel caractère de "mot" : une lettre majuscule ou minuscule, un chiffre ou le caractère _.
\S Inverse de \s. \W Inverse de \w.
Expressions régulières : options
i Insensible à la casse. s Le point reconnaît les sauts de ligne.
m La chaîne est multilignes. e Du code PHP peut être évalué dans le remplacement.
U Rend tous les quantificateurs non gourmands. x Permet de commenter les modèles.
